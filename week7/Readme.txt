В базе данных хранятся данные про студентов курсов SoftGroup (имя, фамилия, телефонный номер, дата рождения, электронная почта).
На страницах вашего web приложения должна быть возможность просмотреть, добавить, обновить, удалить данные о студентах.
Используем PostgreSQL, Hibernate, JSP, Servlet.
=================================================================

Software requirements:
1.Maven
2.Postgresql
3.Hibernate
4.JUnit4 4
5.Tomcat

How To:
Init DB by creating propriate database and needed tables.
All commands and instruction for database can be found at "softgroup_students.backup".
